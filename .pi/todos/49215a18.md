## Description
Several strings use counts/plurals that need proper localization handling. Different languages have different plural rules (e.g., Russian has 3 forms, Arabic has 6).

## Strings Needing Plural Handling
1. **ConversationListView.swift:134** — `Text("\(summary.messageCount) MSG")` → needs plural form ("1 message" vs "5 messages")
2. **FilesView.swift:187** — `Text("\(count) items")` → needs plural form ("1 item" vs "3 items")
3. **DebugPanelView.swift:80** — `Text("\(console.errorCount) ERR")` → needs plural form
4. **CodeEditorView.swift:42** — `Text("\(content.components(separatedBy: "\n").count) L")` → line count plural

## Approach Options
- **Option A**: Use `.stringsdict` files with `NSStringLocalizedFormatKey` for each plural string
- **Option B** (Preferred): Use Swift's automatic grammar agreement with `String(localized:)` and `inflect: true` or use `Text("^[\(count) item](inflect: true)")`

## Example
```swift
// Before
Text("\(count) items")

// After — automatic grammar agreement
Text("^[\(count) item](inflect: true)")
```
