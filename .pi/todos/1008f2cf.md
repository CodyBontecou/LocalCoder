## Description
Create a `Localizable.xcstrings` String Catalog file in the Xcode project. This is the modern (Xcode 15+) approach to localization.

## Steps
1. Add a new String Catalog file (`Localizable.xcstrings`) to the `LocalCoder` target
2. Update `knownRegions` in `project.pbxproj` to include target languages
3. Build the project — Xcode will auto-extract `Text("...")` literals as localization keys
4. Verify all 88+ strings appear in the catalog

## Notes
- SwiftUI `Text("...")` literals are localization keys by default — they auto-lookup in `Localizable.strings`/`.xcstrings`
- This is the foundation all other localization tasks depend on
